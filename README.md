\# Selenium Pytest Automation Framework

This program is to Automate a Website using Selenium Webdriver to Loging into Account and show 2 products



Automation framework built using Python, Pytest, and Selenium WebDriver

following the Page Object Model design pattern.



Tech Stack

\- Python

\- Pytest

\- Selenium WebDriver



Project Structure

PageObject/   - Page Object classes

PythonE2ETestSelenium/   - Test cases

Data/    - Test data





\## How to Run

pip install -r requirements.txt

pytest
